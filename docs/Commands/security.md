# Bảo mật

:::info Ghi chú

- `<>` là **bắt buộc**
- `[]` là **không** bắt buộc

:::

| Lệnh     | Cách dùng                                  | Mô tả     |
| -------- | ------------------------------------------ | --------- |
| login    | `/login <mật khẩu>`                        | Đăng nhập |
| register | `/register <mật khẩu> <nhập lại mật khẩu>` | Đăng ký   |
