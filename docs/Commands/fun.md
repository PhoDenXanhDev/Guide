# Vui

:::info Ghi chú

- `<>` là **bắt buộc**
- `[]` là **không** bắt buộc

:::

| Lệnh      | Cách dùng                                  | Mô tả                            |
| --------- | ------------------------------------------ | -------------------------------- |
| luật chơi | `/taixiu luatchoi`                         | Hiện luật chơi                   |
| cược      | `/taixiu cuoc <tai hoặc xiu> <lượng tiền>` | cược tiền                        |
| thông tin | `/taixiu thongtin`                         | Hiện thông tin về các lượt trước |
